# Welcome to Quest01
***

## Task

The goal of this quest is to learn the basics of the C programming language.
The challenge is understanding core programming concepts such as:

Variables and data types
Conditions (if / else)
Loops (while)
Functions and return values
Character handling with ASCII
Compilation with gcc
Basic output using printf() and write()

Each exercise introduces a small but important concept used in real software development.
---


## Description

This project was completed by solving multiple beginner-level C exercises.

Topics covered in this quest:

Compiling a C program with gcc
Creating integer, character, and string variables
Using increment and decrement operators
Writing conditional statements
Combining multiple conditions with logical operators
Creating and calling functions
Passing parameters to functions
Returning values from functions
Using loops to repeat actions
Recreating standard C library functions such as:
isalpha
isdigit
islower
isupper
isspace
abs

The project also introduces low-level output using the write() system call.
---


## Installation

Clone the repository and compile the exercises using gcc.

Example:

gcc -o my_first_compilation my_first_compilation.c

For other exercises:

gcc -o program_name program_name.c
---


## Usage

Run the compiled executable from the terminal.

Example:

./my_first_compilation

Example output:

my_first_compilation.

Another example:

./my_print_alphabet

Output:

abcdefghijklmnopqrstuvwxyz
